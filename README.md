# 🐧 Linux Documentation & Labs

> Repositório central que reúne, organiza e documenta guias de instalação, configurações de infraestrutura e laboratórios práticos baseados em distribuições Linux.

Este repositório funciona como a minha base de conhecimento (*Knowledge Base*) para administração de sistemas. Aqui estão centralizados os links para as documentações oficiais e os diretórios locais contendo minhas anotações técnicas, arquivos de configuração (*dotfiles*) e scripts de automação para cada ecossistema.

## 🎯 Objetivo

Centralizar referências oficiais e documentar de forma prática o processo de provisionamento, endurecimento (*hardening*) e manutenção de servidores Linux utilizados em ambientes de Infraestrutura, DevOps, Ciência de Dados e IA.

## 📚 Conteúdo

Este repositório foi estruturado para conter:

- 🗺️ Guias de instalação limpa voltados para servidores (*headless* / CLI pura).
- ⚙️ Arquivos de configuração essenciais (SSH, Firewalls, Webservers, Docker daemon).
- 🚀 Scripts em Shell (Bash) para automação de tarefas cotidianas e pós-instalação.
- 💡 Notas sobre otimização de performance para hardware antigo e gerenciamento de recursos.
